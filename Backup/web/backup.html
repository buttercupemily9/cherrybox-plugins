Backup & restore

Archives config.json, the main database, download history, and support-app metadata into a hashed .box zip file. Backup files are stored inside the Backup plugin folder by default. Configure the folder below or schedule automatic backups on the Tasks tab ("Create backup").

Stored backups

CherryBox scans the backup folder for valid .box files each time this list loads. You can copy archives into the folder manually, then click Refresh.

Import backup

Upload a .box file from another CherryBox server and save it in stored backups.

Restore from file

Upload a .box backup and apply it immediately. This replaces current settings and databases.
